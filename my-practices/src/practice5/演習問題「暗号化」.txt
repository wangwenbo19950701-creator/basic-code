1) シ`ザ`暗号はアルファベットを３つずつ大きい方向へずらす暗号法です。たとえば、AはDになります。CARはFDUになります。
またずらしたためにZより大きくなってしまった龊悉Aにもどります。たとえばZはCになります。WXYZはZABCになります。
以下のように、最初に文字列を宣言してアルファベットの大文字で作った文字列で初期化し、その文字列のシ`ザ`暗号を作るプログラムを作りなさい。
そして、そのプログラムで"BEAUTIFUL","WINDOWS","ZUCKERBERG"の三つの文字列が例のように暗号化されることを_Jしなさい。
この例ではクラス名をCaesarCryptとしていますが、クラス名を自分でQめてもYです。

例）文字列の宣言と初期化
public class CaesarCrypt {
      public static void main(String[] args) {
            String src = "ABC";
            // ここから暗号化プログラム
      }
}

例）
文字列             暗号
BEAUTIFUL         EHDXWLIXO
WINDOW            ZLQGRZ
ZUCKERBERG        CXFNHUEHUJ              

2)シ`ザ`暗号を秃扭工毳抓恁哎楗啶蜃鳏辘胜丹ぁＧ同、最初に文字列を宣言してアルファベットの大文字で作った文字列で初期化し、
その文字列のシ`ザ`暗号を秃扭工毳抓恁哎楗啶蜃鳏辘胜丹ぁ
そして、そのプログラムで"PDALFR","DULCRQD","FLQGHUHOOD"の三つの暗号が例のように秃扭丹欷毪长趣虼_Jしなさい。

例）
暗号
PDALFR            MEXICO
DULCRQD           ARIZONA
FLQGHUHOOD        CINDERELLA

3)シ`ザ`暗号ではシフトするが3とQまっていましたが、これを任意のint型整数（これをシフトと呼びます）にできるようにして暗号と秃扭蛐肖staticメソッドを持つを作りなさい。クラス名は自分でQめてYです。

4)前のクラスを使って、あなたの名前をアルファベット大文字にしたものを文字列涫に代入し、
あなたのQ生日（12月15日なら1215、９月３日なら93、４月15日なら415）をint型整数涫に代入しなさい。
次にあなたの名前を代入した文字涫の真ん中（文字列Lを2で割った整数の位置）の文字を整数化し、
Q生日から作った整数涫に加えてこれをシフトとして暗号を生成するプログラムを作りなさい。

例）
①アルファベット大文字で表したあなたの名前
BUNICHIROHFUJII
②Q生日整数
1215
③シフト
1215 + 82('R'の) = 1297
④暗号
YRKFZEFOLECRGFF
